Fiscal Monitor Agent
=====================

Положите сюда архив агента: fiscal-agent-linux.tar.gz

После добавления файла он будет доступен по адресу:
https://fiscaldrive.sbg.network/downloads/fiscal-agent-linux.tar.gz

Структура архива:
  fiscal-agent/
    install.sh      - скрипт установки
    uninstall.sh    - скрипт удаления
    fiscal-agent.sh - основной скрипт агента
    config.json     - пример конфигурации
    README.md       - документация
